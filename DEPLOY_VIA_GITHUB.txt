Problem: Build fails
Solution: 
  - Check logs: heroku logs --tail -a YOUR-APP-NAME
  - Verify package.json has correct Node version
  - Ensure Procfile exists

Problem: App crashes after deploy
Solution:
  - Check if JWT_SECRET is set: heroku config -a YOUR-APP-NAME
  - View error logs: heroku logs --tail -a YOUR-APP-NAME
  - Restart app: heroku restart -a YOUR-APP-NAME

Problem: Can't connect to GitHub
Solution:
  - Use Heroku Dashboard method (easier)
  - Check GitHub repository is public or Heroku has access

Problem: Database issues
Solution:
  - Remember SQLite resets on each deploy
  - Check DEPLOYMENT.md for PostgreSQL migration guide
